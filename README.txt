Prerequisite: Java 17 

Steps to Run Rest Client:
1. Execute "run.bat"
   Note: if you do not execute restclient-3.0.jar using the bat file,"run.bat" or from a command prompt (start java -jar restclient-3.0.jar) 
         then the process will be opened in the background and you will need to close the application from the task manager  )
2. Wait for command prompt to open and server to be started: "Started RestClientApplication in 2.098 seconds (JVM running for 2.603)"
3. Open browser to the following address: http://localhost:8888 and following instructions in the GUI
